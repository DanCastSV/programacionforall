Introducción a Python

En Python, el "Hola, Mundo" es el programa más simple que puedes escribir. Utiliza la función print para mostrar un mensaje en la consola.

print("Hola, Mundo")

En Python, puedes declarar variables sin necesidad de especificar su tipo. Los tipos de datos más comunes incluyen:

  • Enteros (int): Números enteros.
  • Cadenas de texto (str): Secuencias de caracteres entre comillas.
  • Flotantes (float): Números con decimales.
nombre = "Juan"
edad = 25
puntos = 10.5

Python ofrece una variedad de operadores, como:

  • Operadores aritméticos (+, -, *, /): Para realizar operaciones matemáticas.
  • Operadores de comparación (==, !=, >, <): Para comparar valores.
  • Operadores lógicos (and, or, not): Para evaluar expresiones booleanas.
a = 5
b = 3
suma = a + b
es_mayor = a > b

Las estructuras de control permiten tomar decisiones y repetir acciones. Las más comunes son:

  • Condicionales (if-elif-else): Ejecutan código según una condición.
  • Bucles (for y while): Repiten un bloque de código.
if edad >= 18:
  print("Eres mayor de edad")
else:
  print("Eres menor de edad")

Listas (list) y tuplas (tuple) son colecciones ordenadas de elementos. Las listas son mutables (puedes modificarlas), mientras que las tuplas son inmutables (no puedes cambiar su contenido).

frutas = ["manzana", "banana", "naranja"]
coordenadas = (3, 4)

Los diccionarios (dict) almacenan datos en pares clave-valor. Puedes acceder a los valores utilizando las claves.

persona = {"nombre": "Ana", "edad": 30}
print(persona["nombre"])

Las funciones te permiten definir bloques de código reutilizables. Puedes pasar argumentos a una función y obtener un valor de retorno.

def saludar(nombre):
  print("Hola, " + nombre)

saludar("Carlos")

Los módulos y bibliotecas son archivos que contienen funciones y variables que puedes usar en tus programas. Puedes importarlos con import para extender las capacidades de Python.

import math
print(math.sqrt(25))

Las excepciones permiten manejar errores de manera controlada. Puedes usar try y except para manejar excepciones y tomar medidas adecuadas cuando ocurren errores.

try:
  resultado = 10 / 0
except ZeroDivisionError:
  print("Error: División por cero")

En POO, puedes definir clases que actúan como plantillas para crear objetos. Los objetos tienen atributos y métodos asociados.

class Coche:
  def __init__(self, marca, modelo):
      self.marca = marca
      self.modelo = modelo

mi_coche = Coche("Toyota", "Corolla")
